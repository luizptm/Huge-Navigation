
###Requirements

* Node.js and npm (You get both when you <a href="https://docs.npmjs.com/getting-started/installing-node">install Node.js</a>.)

### HOW TO RUN

App can be run by this following commands:

~ npm install
~ npm start

### DOCUMENTATION

The documentation is available on resources/doc.pdf.

The javascript documentation has generated by jsDoc and is available on docs folder.

To generate then, run:
~ jsdoc public/scripts/main.js -d doc
Or:
~ grunt

See: https://github.com/jsdoc3/jsdoc

### TESTS

* Generate Mocha: https://github.com/generate/generate-mocha
* Mocha with Chai: Unit tests
* Sinon: mock JSON request.
* Karma: execute them.
Tests can be run by this following command:

~ npm install --global generate generate-mocha (optional)
~ npm install --only=dev
~ npm test --force
